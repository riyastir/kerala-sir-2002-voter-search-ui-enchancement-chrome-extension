This Chrome extension makes the **Kerala Voter Search** dropdowns
searchable --- in **Malayalam**, **English**, or **Roman Malayalam** ---
with full wildcard and numeric support.\

Constituency)**, and **Booth**, adding a live search box for easier
selection.

## ü™Ñ How to Use

1.  Install [**Google Input
    Tools**](https://chromewebstore.google.com/detail/google-input-tools/mclkkofklkfljcocdinagocijmpgbhab?hl=en)\
2.  Add **Malayalam** as one of your input languages\
3.  Install this extension (**Malayalam Dropdown Search Enhancer**)\
4.  Visit
    [**https://www.ceo.kerala.gov.in/voter-search**](https://www.ceo.kerala.gov.in/voter-search)\
5.  When dropdowns (District, LAC, Booth) appear, you'll see a search
    box at the top\
6.  Type in **Malayalam** or **English** --- results appear instantly
